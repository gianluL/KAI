This is KAI.

To set it up under linux (debian-based), please use write "./setup.sh"

This repository includes the ./BOT folder for bot-related python code and ./API for api-related python code. Please modify the ".env" files in both directories to match your tokens, endpoints and credentials.

If the machine is already set up, you can simply run "./start.sh"

N.B.
Please ensure you modified the .env files according to your needs and credentials!!
